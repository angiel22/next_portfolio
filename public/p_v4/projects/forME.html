forME

Role Collaboration Duration
Fullstack Developer Team of 4 March 2023

Scenario

Easier interface and makes more sense for the project we wanted to create. Especially with trackers and medication. Spread awareness on the taboos of mental health, birth control, and taking medication. Create an app that’s practical, informative, and user friendly.

Solution

forME is a mobile app built for women to be educated about the medications / birth control going into their body. forME allows users to search for different medications and birth control options that are currently available. forME has a tracker function that allows users to track their medication and habits
